LoginPage

LoginPage class

A page that allows the user to log in by providing a Home ID, username, and password.

Fields:

  • homeIdController (TextEditingController): Controller for Home ID input.
  • usernameController (TextEditingController): Controller for username input.
  • passwordController (TextEditingController): Controller for password input.
  • errorMessage (String?): Stores error messages to display when login fails.

Method:

  • attemptLogin():
    • API: Authentication Service
    • Endpoint: POST https://services.iittp.ac.in/var/homes/{homeId}/get-key/
    • Request Body: (x-www-form-urlencoded)
      • username (String): User’s username.
      • password (String): User’s password.
    • Success Response (200 OK): 
      { "api_key": "<generated_api_key_string>" }
      • Stores api_key in local database and SharedPreferences.
      • Navigates to HomePage.
    • Error Responses (4xx): 
      { "error": "<error_message>" }
      • Parses and displays the error message.
    • Exceptions: Connection or JSON parsing errors are caught and displayed as "Connection or parsing error occurred".
